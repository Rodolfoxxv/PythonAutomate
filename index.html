Automação de BI: Geração de Dados Complexos - Parte 1

Este projeto é a primeira parte de uma série focada na automação de Business Intelligence (BI). O objetivo é desenvolver um sistema robusto que possa gerar e armazenar dados de consultas complexas no Google Cloud Platform (GCP) usando Terraform. Esses dados serão posteriormente utilizados para análises avançadas e tomada de decisões estratégicas.
